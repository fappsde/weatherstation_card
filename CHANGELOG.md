# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial HACS setup with hacs.json and info.md
- Complete TypeScript-based card implementation
- Real-time weather data display for Ecowitt WS90
- Support for temperature, humidity, pressure
- Wind speed and direction display
- Rainfall data with rate
- UV index with color-coded levels
- Solar radiation display
- Customizable show/hide options for each sensor
- Responsive grid layout
- Theme support following Home Assistant styles
- Build system with Rollup
- Comprehensive test suite with Jest
- ESLint and Prettier for code quality
- GitHub Actions CI/CD workflows
- Automated release workflow
- HACS validation workflow
- Dependabot configuration
- Issue and PR templates
- Comprehensive documentation
